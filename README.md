# JV-FE react-hook-template

## 命令行支持:
   
`npm run dev`
   
    以开发模式启动项目,默认在8080端口,在package.json的port字段可以修改启动端口.

`npm run build`

    构建生成环境适用的build安装包.

`npm run analyze`

    构建生成环境适用的build安装包,并同步生成依赖分析报告图.

`npm run test`

    执行 /src/test/unit_testing 目录下的单元测试用例.


`npm run doc`

    生成开发文档
